# 功能摘要 (Features Summary)

## 已實現的功能

### 1. 區域選取放大 (Region Selection Zoom)
- **啟用方式**：點擊工具列的「區域放大」按鈕或使用快捷鍵 Ctrl+Z
- **操作方式**：
  1. 啟用區域放大模式後，滑鼠游標變為十字形
  2. 在影像上按住滑鼠左鍵並拖曳，選取想要放大的區域
  3. 放開滑鼠後，會彈出對話框要求輸入放大倍率（範圍 0.1-10.0）
  4. 確認後，選取的區域會以指定倍率放大並顯示在新視窗中

### 2. 放大視窗 (Zoom Window)
- **特點**：
  - 獨立的新視窗顯示放大後的影像
  - 使用捲軸處理超出視窗範圍的影像
  - 提供完整的工具列功能

### 3. 畫筆功能 (Brush/Drawing Tool)
- **啟用方式**：在放大視窗中點擊「畫筆模式」按鈕
- **功能**：
  - 可在放大的影像上繪製線條
  - 支援自訂筆刷顏色（透過色彩選擇器）
  - 支援調整筆刷寬度（1-50 像素）
  - 繪製時游標顯示為十字形
  - 繪製的內容會直接修改影像資料

### 4. 另存新檔 (Save As)
- **啟用方式**：在放大視窗中點擊「另存新檔」按鈕
- **功能**：
  - 支援多種影像格式：PNG、JPEG、BMP
  - 儲存的影像包含所有繪製的標註內容
  - 成功儲存後顯示確認訊息

## 使用流程範例

### 完整操作流程：
1. 開啟應用程式：`./ImageProcessor`
2. 載入影像檔案：檔案 → 開啟檔案 (Ctrl+O)
3. 啟用區域放大：工具 → 區域放大 (Ctrl+Z)
4. 在影像上拖曳選取想要放大的區域
5. 輸入放大倍率（例如：2.5）
6. 在新開啟的放大視窗中：
   - 點擊「畫筆模式」啟用繪圖
   - 點擊「選擇顏色」選擇筆刷顏色
   - 調整「筆寬」數值設定筆刷大小
   - 在影像上拖曳滑鼠進行標註
7. 完成標註後，點擊「另存新檔」儲存影像
